# 実装工程

このファイルは、仕様駆動開発の**実装工程**で作成される実装記録です。

## 実装工程の原則

仕様駆動開発の実装工程では、以下の原則に基づいて実装を進めます：

- **小さな変更単位でコミット**：レビュー負荷を軽減
- **テストコードもAIに生成させる**：CI/CDで自動検証
- **人が必ず確認する**：AI出力を鵜呑みにしない（ヒューマン・イン・ザ・ループ）

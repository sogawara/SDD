# 書籍の正誤表

本書『仕様駆動開発 実践入門』の発売後に判明した誤植や修正事項を記録します。

## このページについて

このページでは、書籍発売後に判明した以下の内容を記録します：

- **誤植**：文字の誤り、表記の不統一など
- **内容の修正**：より正確な表現への修正、追記など
- **技術的な更新**：ツールのバージョンアップに伴う手順の変更など

## 誤植・修正事項の報告方法

誤植や修正事項を発見された場合は、以下の方法でお知らせください：

- **GitHub Issues**：SDDリポジトリ（https://github.com/elvezjp/SDD）のIssuesで報告
- **Pull Request**：修正案がある場合は、Pull Requestで提案
